The Missing Semester

2023-01-04

  • Shell

    • echo $PATH

    • which echo 查找要运行的“echo”位置

    • macos、linux路径是斜杠，而windows是反斜杠

    • pwd 查看当前路径 ls 列出当前路径内容

    • cd 改变路径 . 当前路径 .. 前一路径

    • ~ 用户文件夹 - 上一个位置

    • ls —help 查看帮助文档（中文版）

    • ~/bin 中文件所有用户均可执行

    • mv 重命名/移动文件位置

    • cp 复制文件

    • rm 删除文件(永久删除，不进回收站) rmdir 删除指定空目录

    • mkdir 创建空目录

    • man ls 查看指令的手册

    • Ctrl+L 清屏终端并回到顶部

    • echo hello > hello.txt 重定向输入输出(如echo到文件中、从文件中echo)

    • cat hello.txt 打印文件内容到终端

    • >> 在输入输出流后面追加内容

    • | 管道，将|前面的输出连接为|后面的输入

    • tail -n5 打印出输入文本的最后五行

    • sudo管理员权限 sudo su Root权限

    • ~/sys 各种硬件配置

  • Shell脚本(本节适用于bash中)

    • foo=bar (中间无空格)将bar赋给变量foo

    • $foo调用“bar” (fish中是set foo bar)

    • $(ls) 调用括号内容(如ls的输出) 如foo=$(pwd) 将pwd输出内容存到foo变量中

    • ‘ ‘ 引用的字符串输出原本字符 如echo ‘Value is $foo’ 输出为$foo

    • “ “ 引用的字符串输出特殊含义 如上面一句输出为foo储存的量

    • $ + 字符

      • $? 判别上一个命令是否产生错误 echo $?

        • 若错误或未执行，返回 1 (理解为未正确执行的报错1)；

        • 若正确，返回 0 (理解为正确执行后的退出代码return 0)

      • $0 正在运行的脚本的名称

      • $# 给命令的参数数量

      • $$ 正在运行的命令的进程ID

      • $@ 进程中的所有参数

    • true 永远真, $? == 0 ； false 永远假，$? == 1

    • && 逻辑且 如果前面真($? == 0) 后面才会执行；只要有一个$? == 1，就不会向后执行

    • || 逻辑或 如果前面假($? == 1) 后面才会执行；只要有一个$? == 0，就不会向后执行

    • ; 连接两端代码，但语句结尾不加分号

    • grep

  • Vim

    • 见手写笔记

Lecture 2

python中的变量赋值(如x = n*n)是绑定性的，后续更改等号右边量的值，不会改变等号左边的值

此时的函数名称和变量名称本质等价，一个绑定到函数上，一个绑定到变量上

也就是说可以将变量绑定到多个语句(函数)上，实现抽象封装

多变量赋值语句中，先同时计算右边的值，再同时赋给左边

用function可以实现想要的功能，python十分注重函数思想

一类函数执行特殊功能，但返回值是None，如print()
