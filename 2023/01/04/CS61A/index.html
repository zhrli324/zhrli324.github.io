Lecture 2

python中的变量赋值(如x = n*n)是绑定性的，后续更改等号右边量的值，不会改变等号左边的值

此时的函数名称和变量名称本质等价，一个绑定到函数上，一个绑定到变量上

也就是说可以将变量绑定到多个语句(函数)上，实现抽象封装

n = 1
x = n*n
print(x)
#此时x为1
n = 3
print(x)
#此时x仍为1，而不是9

多变量赋值语句中，先同时计算右边的值，再同时赋给左边

用function可以实现想要的功能，python十分注重函数思想

一类函数执行特殊功能，但返回值是None，如print()

>>> a = print(2)
>>> 2
>>> print(a)
>>> None
